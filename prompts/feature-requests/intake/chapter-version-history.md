The system already snapshots chapter versions every time Verity touches a file. But there's no way to browse that history or compare versions. After a heavy revision pass, I want to see what changed — either a side-by-side diff of two versions or a visual timeline of how a chapter evolved across multiple agent passes.

This should live in the FilesView as a new "History" tab. Select a chapter, see a list of versions with timestamps and which agent/action created them (e.g., "Auto-draft", "Manual edit", "Verity revision pass", "Sable copy-edit"). Click two versions to see a diff. Optionally, revert to an older version with one click.

Especially useful during revision when you're assessing whether a scene actually improved, or during copy-edit to verify that Sable didn't accidentally change the voice.
