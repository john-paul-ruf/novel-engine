Series authors reuse characters across books. Today you have the Series Bible for series-level plot notes, but characters are trapped in individual Character Profiles (feature request). A global Character Registry would live outside any single book and let you: define a character once, reference them across multiple books, track their arc across volumes.

When creating a new book in a series, add existing characters from the registry with a click. The character's existing profile — appearance, background, established relationships — populates automatically. You can then add book-specific additions (new relationships, conflicts introduced in this volume, status changes). Agents see the global character profile plus book-specific additions.

Essential for multi-book series where reader confusion about character continuity is a real risk.
