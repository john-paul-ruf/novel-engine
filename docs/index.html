The Seven Agents.

Not one general AI — a team of specialists. Each agent has a distinct role, perspective, and editorial voice.

Spark
The Ideator
Explores your idea through conversation, then produces a full pitch card — premise, themes, characters, emotional engine, opening hook. Brainstorms in the Pitch Room before you commit to a book.
Verity
The Ghostwriter
The only agent that writes prose. Drafts chapters using your captured voice profile, builds the scene outline and story bible, implements revision changes. Phase-specific sub-prompts for scaffolding, drafting, revision, and mechanical fixes. Integrated with the motif ledger.
Ghostlight
The First Reader
Reads the manuscript cold — no notes, no context, no outline — and reports the unfiltered reader experience. Your first honest audience.
Lumen
The Developmental Editor
Seven-lens structural analysis: protagonist arc, supporting cast, pacing, scene necessity, theme, narrative logic, and a revision roadmap. Deep structural and thematic assessment.
Sable
The Copy Editor
Line-level precision — grammar, style consistency, mechanical errors. Produces the audit report, maintains the style sheet, and tracks motif consistency across the manuscript.
Forge
The Task Master
Synthesizes reader and dev reports into a prioritized, phased revision plan with session-by-session execution prompts for Verity. Builds the actual revision queue.
Quill
The Query Crafter
Audits the final manuscript and prepares publication metadata — title, description, keywords, BISAC categories, back-cover copy. Your publishing-ready package.

Key Features.

Everything you need to go from idea to published manuscript.

Creative

Pitch Room

Free brainstorming space with Spark. Explore story ideas before committing. Shelve, restore, or convert pitches into full book projects with one click.

Voice

Voice Profile

A guided interview captures your sentence rhythm, vocabulary, dialogue style, emotional temperature, and tonal anchors. Verity writes in YOUR voice — not generic AI prose.

Automation

Auto-Draft

One click starts a loop where Verity writes chapters in sequence. Per-chapter quality audits catch issues in real-time. Motif audits run every 3 chapters to keep the motif ledger accurate.

Quality

Verity Audit Pipeline

Automated quality assurance after every agent interaction. A Sonnet-powered audit agent scans for editorial narration, anti-patterns, voice drift, and continuity errors. Fixes are applied automatically.

Tracking

Motif Ledger

Seven-panel structured tracker for recurring literary elements: motif systems, character entries, structural devices, foreshadowing, flagged phrases, and audit logs. Stored as JSON per book.

Revision

Revision Queue

Forge builds structured revision plans. The queue parses them into sessions and executes them through Verity with four modes: manual, auto-approve, auto-skip, and selective. Approval gates at every checkpoint.

Version Control

File Version History

Every file written by an agent or saved by the user is automatically versioned in SQLite. Visual diff viewer, one-click revert, SHA-256 dedup, and source tracking (user, agent, or revert).

Extensible

Multi-Model Providers

Ships with Claude Code CLI as the built-in provider. Add any OpenAI-compatible endpoint — OpenAI, Ollama, LM Studio, vLLM, Groq, Together. Provider management, status checking, and model routing from Settings.

Export

Build & Export

Assembles chapters via Pandoc into Markdown, DOCX, and EPUB. Download All exports a ZIP. Catalog Export creates a ZIP of all books for backup or transfer.

For Writers.

You bring the story. The agents build it into a book.

What You Need

  • macOS, Windows, or Linux
  • Claude Code CLI — installed, authenticated, and on your PATH. The CLI handles all AI calls; no API keys needed in the app.
  • Pre-built installers available from Releases, or build from source with Node.js 20+.
  • Optional: OpenAI-compatible providers for additional model support (requires their own API keys).

Quick Start

  • Install Novel Engine (download or build from source)
  • Launch — the Onboarding Wizard detects your Claude Code CLI
  • Set up your Author Profile and Voice Profile — the AI learns YOUR writing voice through an interview process
  • Create a book and start with Spark in the Pitch Room
  • Work through the pipeline — pitch, scaffold, draft, read, revise, copy-edit, build

Voice Profile

Before Verity writes a single word, you establish a Voice Profile — a detailed document capturing your sentence rhythm, vocabulary register, dialogue style, emotional temperature, interiority depth, punctuation habits, and tonal anchors. Verity conducts a guided interview to extract your authentic voice. The result: prose that sounds like you wrote it.

Export Formats

The Build phase assembles all chapters via Pandoc and exports to:

  • Markdown (.md) — concatenated with title page
  • DOCX (.docx) — Word-compatible
  • EPUB (.epub) — e-reader ready with cover image support

Download All exports a ZIP archive of every format. Catalog Export creates a full backup of all books.
