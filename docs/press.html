The Pitch.

Novel Engine is to book-writing what a build system is to software. It's not a chatbot. It's not an AI ghostwriter. It's an editorial production pipeline where the human author retains creative authority and seven specialized AI agents serve as a professional editorial team — each with a distinct role, perspective, and voice.

The author drives every creative decision. The agents pitch story concepts, scaffold outlines, draft prose in the author's captured voice, cold-read the manuscript, perform structural analysis, plan revisions, copy-edit, and compile the final manuscript into export-ready formats. The pipeline takes a book from pitch to polished manuscript in 14 structured phases, each with explicit completion gates the author confirms before advancing.

Five published novels on Amazon were built with this system. A dual-AI evaluation (Claude + GPT) scored them 7.0–9.4 out of 10. This is not a prototype — it ships real books.

What Makes This Different.

Build system, not chatbot

14-phase pipeline with completion gates, not freeform prompting. Each phase has defined inputs, outputs, and a designated agent.

7 specialized agents

Not one general AI, but a team with distinct roles: ideation, ghostwriting, cold reading, structural editing, copy editing, task planning, and publishing.

Voice capture

Writes in the author's voice via a detailed profile interview. Sentence rhythm, vocabulary, dialogue style, emotional temperature — all captured and applied.

Local-first

Everything runs on your machine. No cloud backend. No API key storage for the built-in provider. Claude Code CLI handles authentication.

Multi-model support

Ships with Claude CLI as the built-in provider. Add any OpenAI-compatible endpoint — OpenAI, Ollama, LM Studio, vLLM, Groq, Together.

Open source

AGPL-3.0 licensed. 130 TypeScript source files. Five clean architecture layers. Fully inspectable and extensible.

Ships real books

5 published novels on Amazon. Dual-AI evaluation scores: 7.0–9.4. Not a tech demo — a working production tool.
