Slide-over panel displaying file version history as a timeline with source badges, relative timestamps, byte sizes. Clicking a version computes and displays the diff. One-click revert with inline confirmation.

Created versionStore Zustand store with paginated history, version selection, auto-diff, and revert. Created DiffViewer component rendering unified diffs with color-coded lines.

Connected VersionService to Electron. 6 new versions:* IPC channels, window.novelEngine.versions preload namespace. Auto-snapshot hooks at 5 capture points. Startup pruning on app launch.

feature Add VersionService implementation with diff computation

Created VersionService implementing all 8 methods of IVersionService. Uses diff package for structured diffs, SHA-256 hashing for dedup.

infra Add database migration and version repository

SQLite migration v2 creating file_versions table with composite indexes. Extended IDatabaseService with 7 new methods for version CRUD.

infra Add domain types and interface for content version control

Added 7 version control types and IVersionService interface with 8 methods. Foundation for the content-version-control feature.

docs Add intake meta-prompt for document-to-session decomposition

Created prompts/meta/intake.md — generic meta-prompt for decomposing feature specs into ordered session prompts with MASTER/STATE tracking.

fix Fix MotifLedgerService data loss: remove auto-writeback, harden JSON repair

The initial JSON repair implementation auto-wrote repaired data back to disk on load, corrupting the structure and causing total data loss. Fixed by removing auto-writeback and rewriting repair to line-by-line structural approach.
